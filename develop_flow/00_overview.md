# 00 計算機アプリ：全体設計

## 目的

HTML、CSS、TypeScriptを使用して、ブラウザ上で動作する計算機を作成する。

Reactやバックエンドは使用せず、Webページの基本的な仕組みを理解する。

## 使用技術

- HTML
- CSS
- TypeScript
- Vite
- Git
- GitHub Pages

## 実装する機能

- 0〜9の数字入力
- 足し算
- 引き算
- 掛け算
- 割り算
- 小数点
- ACによる全消去
- 1文字削除
- イコールによる計算
- 0除算時のエラー表示
- スマートフォン対応

## 開発順序

1. Viteの環境を構築する
2. HTMLで画面の構造を作る
3. CSSで計算機らしく配置する
4. 数字入力を実装する
5. 計算状態を管理する
6. 四則演算を実装する
7. 補助機能とエラー処理を追加する
8. 動作テストを行う
9. GitHub Pagesで公開する

## 完成条件

- 四則演算が正常に動作する
- 小数を入力できる
- 0で割った場合にエラーが表示される
- ACを押すと初期状態へ戻る
- スマートフォンでも操作できる
- GitHub上にソースコードがある
- 公開URLから誰でも利用できる
