# 02 HTMLで計算機の構造を作る

## 目的

HTMLを使って、計算機に必要な表示画面とボタンを配置する。

この段階では、デザインや計算処理は実装しない。  
Webページを構成する各要素の役割を理解することを目的とする。

## 今回の完成イメージ

以下の要素をブラウザに表示する。

```text
┌─────────────────┐
│               0 │
├────┬────┬────┬────┤
│ AC │ ←  │  % │ ÷  │
├────┼────┼────┼────┤
│  7 │  8 │  9 │ ×  │
├────┼────┼────┼────┤
│  4 │  5 │  6 │ −  │
├────┼────┼────┼────┤
│  1 │  2 │  3 │ ＋ │
```

必要な文字やボタンがすべて表示されれば完了とする。

## HTML・CSS・TypeScriptの役割

今回のアプリでは、それぞれ次の役割を持つ。

```text
HTML
→ 表示画面やボタンなど、ページの構造を作る

CSS
→ 色、大きさ、配置など、ページの見た目を整える

TypeScript
→ ボタンを押したときの入力や計算処理を行う
```

今回はHTMLの構造だけを作る。

## 1. Viteの初期コードを確認する

プロジェクト作成直後の `index.html` には、次のような記述がある。

```html
<div id="app"></div>
<script type="module" src="/src/main.ts"></script>
```

`src/main.ts` の処理によって、`id="app"` の中にViteの初期画面が作られている。

今回は、TypeScriptから画面全体を生成するのではなく、HTMLファイルに計算機の構造を直接記述する。

## 2. `src/main.ts`を整理する

Viteの初期画面を作成しているコードを削除する。

この段階では、CSSを読み込む処理だけを残す。

```typescript
import "./style.css";
```

計算処理は、後の工程で追加する。

## 3. `src/style.css`を整理する

Viteが最初から用意しているCSSをすべて削除する。

この段階では、次の最低限の設定だけを書く。

```css
* {
  box-sizing: border-box;
}

body {
  margin: 0;
}
```

### `box-sizing: border-box`

要素の幅や高さに、余白や枠線を含めて計算する設定。

後でボタンの大きさを揃えやすくなる。

### `margin: 0`

ブラウザが最初から設定しているページ外側の余白を削除する。

## 4. HTMLの言語を日本語にする

`index.html` の先頭付近にあるHTML要素を確認する。

```html
<html lang="en">
```

これを日本語に変更する。

```html
<html lang="ja">
```

`lang`属性は、このページで主に使用されている言語をブラウザや検索エンジンに伝える。

## 5. ページタイトルを変更する

`head`要素内の`title`を確認する。

計算機アプリであることが分かるタイトルに変更する。

```html
<title>計算機</title>
```

`title`の内容は、ブラウザのタブに表示される。

## 6. 計算機全体を囲む

`body`要素の中に、計算機全体を表す`main`要素を作る。

```html
<main class="calculator">
</main>
```

### `main`

ページの中心的な内容を表すHTML要素。

### `class="calculator"`

CSSから計算機全体を指定するための名前。

同じclass名は複数の要素に付けることができる。

## 7. 表示画面を作る

`main`要素の中に、入力中の数字や計算結果を表示する要素を作る。

```html
<output id="display">0</output>
```

### `output`

計算結果など、処理の結果を表示するためのHTML要素。

### `id="display"`

後でTypeScriptから表示内容を書き換えるために使用する。

`id`は、同じページ内で重複させない。

### 初期値の`0`

計算機を開いた直後に表示する数字。

## 8. ボタン全体を囲む

表示画面の下に、すべてのボタンを囲む要素を作る。

```html
<div class="buttons">
</div>
```

後の工程では、この`buttons`にCSS Gridを設定して、ボタンを4列に並べる。

## 9. 数字ボタンを作る

数字ボタンは、次のような形式で作成する。

```html
<button type="button" data-number="7">7</button>
```

### `button`

クリックできるボタンを表すHTML要素。

### `type="button"`

フォーム送信ではなく、通常の操作ボタンとして使用することを示す。

### `data-number="7"`

TypeScriptから、どの数字ボタンが押されたかを取得するための情報。

### ボタン内側の`7`

画面上で利用者に表示される文字。

同じ形式で、`0`から`9`までの数字ボタンを作成する。

## 10. 演算子ボタンを作る

演算子ボタンは、次のような形式で作成する。

```html
<button type="button" data-operator="+">＋</button>
```

画面には全角の`＋`を表示し、プログラム内部では半角の`+`を使用する。

以下の演算子ボタンを作成する。

```text
+
-
*
/
```

画面上の表示は、次のようにしてよい。

```text
＋
−
×
÷
```

## 11. 操作ボタンを作る

次の操作ボタンを作成する。

```text
AC
1文字削除
%
小数点
=
```

用途ごとに`data-action`属性を付ける。

例：

```html
<button type="button" data-action="clear">AC</button>
```

```html
<button type="button" data-action="delete">←</button>
```

```html
<button type="button" data-action="percent">%</button>
```

```html
<button type="button" data-action="decimal">.</button>
```

```html
<button type="button" data-action="equals">＝</button>
```

`data-action`の名前は、後でTypeScriptから処理を判別するために使用する。

## 12. ボタンを並べる順番

`buttons`要素の中に、次の順番でボタンを書く。

```text
AC  ←  %  ÷
7   8   9  ×
4   5   6  −
1   2   3  ＋
0   0   .  ＝
```

最後の行の`0`は、後のCSSで横幅を2列分にする予定である。

そのため、HTMLでは`0`ボタンを1つだけ作成する。

```text
0  .  ＝
```

## 13. `script`要素を確認する

`body`要素の最後に、次の記述があることを確認する。

```html
<script type="module" src="/src/main.ts"></script>
```

この記述により、`src/main.ts`がブラウザから読み込まれる。

削除しないように注意する。

## HTMLの構造

最終的な親子関係は、次のようになる。

```text
body
└── main.calculator
    ├── output#display
    └── div.buttons
        ├── button
        ├── button
        ├── button
        └── ...
```
